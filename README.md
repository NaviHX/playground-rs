# Playground-rs

This is a custom helper library that includes the following modules:

## Module List

- **algebra**
    - **qpow**
    - **modular**: Modular plus / multiply.
- **data_structure**
    - **trie**
- **matrix**
- **matrix_like**: Provides an interface for `matrix` containers.
- **parser_combinator**
